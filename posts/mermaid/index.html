Mermaid

Richard Hernandez Apr 25, 2021 2021-04-25T23:32:00-04:00
Apr 27, 2021 2021-04-27T00:20:02-04:00 1 min

This tool allows the creation of diagrams using markdown and their own syntax. an example is shown below:

classDiagram
    Animal <|-- Duck
    Animal <|-- Fish
    Animal <|-- Zebra
    Animal : +int age
    Animal : +String gender
    Animal: +isMammal()
    Animal: +mate()
    class Duck{
      +String beakColor
      +swim()
      +quack()
    }
    class Fish{
      -int sizeInFeet
      -canEat()
    }
    class Zebra{
      +bool is_wild
      +run()
    }
